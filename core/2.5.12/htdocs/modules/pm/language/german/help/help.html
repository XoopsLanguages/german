Hilfe: Private Nachrichten

BESCHREIBUNG

Das PM-Modul dient dem privaten Nachrichtenaustausch zwischen Benutzern.

INSTALLATION/DEINSTALLATION

Es sind keine besonderen Maßnahmen erforderlich. Folgen Sie dem Standard-Installationsprozess – entpacken Sie den Ordner /pm in das Verzeichnis ../modules. Installieren Sie das Modul über Admin -> Systemmodul -> Module.

Detaillierte Anweisungen zur Installation von Modulen finden Sie im XOOPS-Betriebshandbuch.

BEDIENUNGSANLEITUNG

So richten Sie dieses Modul ein:

  • Sie erreichen Ihr Postfach direkt über die Startseite oder über den Link „Zum Modul“ im Adminbereich des PM-Moduls.
  • Konfigurieren Sie die Einstellungen des Moduls (siehe „Einstellungen“) und optional den PM-Block, falls Sie diesen verwenden möchten (siehe „Blöcke“).
  • Sie können Nachrichten massenhaft über den Tab „Nachrichten bereinigen“ löschen.
