Hilfe: Profil

BESCHREIBUNG

Das Profil-Modul dient zur Verwaltung benutzerdefinierter Profilfelder.

INSTALLATION/DEINSTALLATION

Keine besonderen Maßnahmen erforderlich. Folgen Sie dem Standard-Installationsprozess – entpacken Sie den Ordner /profile in das Verzeichnis ../modules. Installieren Sie das Modul über Admin -> Systemmodul -> Module.

Detaillierte Anweisungen zur Modulinstallation finden Sie im XOOPS-Benutzerhandbuch

BEDIENUNGSANLEITUNG

Zur Einrichtung dieses Moduls müssen Sie:

  • Die Einstellungen des Moduls konfigurieren (siehe „Einstellungen“) und optional den Profil-Block aktivieren (siehe „Blöcke“).
  • Vorhandene Kategorien bearbeiten oder neue hinzufügen.
  • Vorhandene Felder bearbeiten oder neue erstellen. Hier können Sie festlegen, in welcher Kategorie die Felder angezeigt werden und ob sie während der Registrierung sichtbar sind.
  • Die Reihenfolge der Registrierungsschritte festlegen.
  • Schließlich können Sie die Rechte für einzelne Felder festlegen – z. B. welche editierbar oder durchsuchbar sind.

Anti-Spam-Maßnahmen

Um Spam-Registrierungen zu minimieren, gehen Sie wie folgt vor:

  • Im Protector-Modul unter „Einstellungen“ ganz nach unten scrollen zur Option:

    „Stop Forum Spam“
    Diese Funktion überprüft POST-Daten mit der Datenbank von www.stopforumspam.com.
    Setzen Sie die Option auf „IP-Adresse sperren (ohne Begrenzung)“.

  • In der Datei /class/captcha/config.php sicherstellen, dass der Modus auf „text“ gesetzt ist:

    return $config = array(
    'disabled' => false,
    'mode' => 'text',
    'name' => 'xoopscaptcha',
    'skipmember' => true,
    'maxattempts' => 10,
    );

  • Im Profil-Modul unter Admin im „Basic Step“:
    /modules/profile/admin/step.php?id=1
    setzen Sie die Option „Nach dem Stop speichern“ auf „Nein“.

  • In den Profileinstellungen die Option „Captcha nach dem zweiten Registrierungsschritt verwenden“ auf „Ja“ setzen (Standardwert).
