Hilfe: Benutzer

Beschreibung

XOOPS stellt integrierte Werkzeuge zur einfachen Verwaltung von Benutzern bereit. Dazu gehören die Möglichkeit, Benutzer nach verschiedenen Kriterien zu suchen, sowie das Versenden von E-Mails und privaten Nachrichten über ein vorlagenbasiertes Nachrichtensystem. Auf Ihrer Website haben Benutzer die Rechte, die der Gruppe zugewiesen sind, der sie angehören.

Jeder Benutzer hat Zugriff auf sein eigenes Profil und kann steuern, welche Informationen für andere sichtbar sind. Außerdem verfügt er über ein Postfach, in dem er private Nachrichten vom Administrator oder von anderen Benutzern empfangen kann. Zusätzlich kann der Benutzer festlegen, bei welchen Änderungen auf der Website er Benachrichtigungen erhalten möchte. Je nach den vom Administrator vergebenen Berechtigungen können Benutzer Inhalte sehen, Inhalte hochladen, Artikel oder Kommentare verfassen, Inhalte bewerten und generell mit ihrer Rückmeldung zur Weiterentwicklung der Website beitragen.

Erweiterte Suche

Die Funktion „Erweiterte Suche“ bietet eine Vielzahl von Optionen, um gezielt nach bestimmten Benutzern zu suchen oder Ihre Mitgliederliste zu filtern. Sie können auch eine vollständige Liste aller Mitglieder abrufen, indem Sie auf „Senden“ klicken, ohne Filter auszuwählen.

Benutzer bearbeiten

In diesem Bereich des Systemmoduls können Sie Benutzer hinzufügen, bearbeiten oder löschen.

Um einen Benutzer zu bearbeiten, suchen Sie ihn anhand des Namens oder anderer Kriterien. Danach können Sie entweder:

i) seine Profildaten ändern oder
ii) ihn löschen – über das zweite Auswahlfeld.

Wenn Sie „Bearbeiten“ auswählen, wird das Profil des Benutzers geöffnet und kann nach Bedarf geändert werden (Hinweis: Sie können das Bearbeitungsformular auch über das Mitgliederverzeichnis oder durch Klick auf den Benutzernamen an anderer Stelle der Website erreichen).
