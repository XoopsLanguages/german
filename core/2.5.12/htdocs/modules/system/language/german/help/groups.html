Hilfe: Gruppen

Beschreibung

XOOPS verfügt über ein integriertes Registrierungssystem – Sie können optional verlangen, dass sich Benutzer auf Ihrer Website anmelden, um Zugriff auf bestimmte Bereiche oder Funktionen zu erhalten. Die meisten Websites erlauben zum Beispiel das Veröffentlichen von Nachrichten oder das Schreiben in Foren nur für registrierte Benutzer.

Der Zugriff und die Verwaltungsrechte auf einer XOOPS-Website werden über ein flexibles Berechtigungssystem geregelt, das auf „Benutzergruppen“ basiert. Die Standardgruppen umfassen anonyme Benutzer, registrierte Mitglieder und Webmaster – Sie können jedoch beliebig viele eigene Gruppen definieren, um Ihre Website besser zu verwalten. Jeder Gruppe kann ein einzigartiger Satz von Zugriffsrechten zugewiesen werden, der bestimmt:

– welche Blöcke sichtbar sind
– auf welche Module zugegriffen werden kann
– welche Module administriert werden dürfen
– welche Bereiche der Systemadministration geändert werden dürfen

Die Namen dieser Standardgruppen können geändert werden, aber sie selbst können nicht gelöscht werden. Weitere Benutzergruppen können wie später beschrieben erstellt und konfiguriert werden.

Eigene Gruppen können über die Gruppenverwaltungsseite erstellt und verwaltet werden.
