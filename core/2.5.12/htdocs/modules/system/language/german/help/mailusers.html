Hilfe: Benutzer per E-Mail kontaktieren

Beschreibung

XOOPS verfügt über ein integriertes Nachrichtensystem, das es Ihnen ermöglicht, eine Nachricht an Ihre gesamte registrierte Benutzerbasis oder an Untergruppen davon zu senden, wobei auch Datumsfilter verwendet werden können. Dies kann in Form einer privaten Nachricht (d.h. über das integrierte Nachrichtensystem von XOOPS) oder per E-Mail erfolgen. Es stehen verschiedene Optionen zur Verfügung, um Untergruppen von Benutzern aus Ihrer Mitgliederbasis herauszufiltern.

Sie können auch vordefinierte Tags verwenden, die es Ihnen ermöglichen, Ihre Nachricht zu personalisieren.
