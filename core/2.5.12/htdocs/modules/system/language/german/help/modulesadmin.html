Hilfe: Module

Beschreibung

Module sind Software-Bausteine, die – basierend auf der XOOPS-Infrastruktur – dem Webmaster ermöglichen, den Benutzern der Website dynamische oder statische Inhalte über eine benutzerfreundliche Weboberfläche bereitzustellen.

Ein großer Teil der Leistungsfähigkeit von XOOPS liegt in der Art und Weise, wie Module verwaltet werden. Der Webmaster kann ein bestimmtes Modul (außer dem Systemmodul – aus naheliegenden Gründen) ganz einfach installieren, deinstallieren, aktivieren oder deaktivieren. Es ist außerdem möglich, Module umzubenennen und ihre Reihenfolge im Navigationsmenü für Endbenutzer festzulegen (sofern sie dort erscheinen sollen). Sie können auch Module aktivieren, ohne sie im Menü anzuzeigen – so sind sie für normale Benutzer unsichtbar.

Zum Beispiel könnten Sie das Nachrichtenmodul so konfigurieren, dass es nicht im Menü erscheint, indem Sie das Häkchen in der Spalte „Menü“ deaktivieren. Sie könnten dann ausgewählten Benutzern eine private Nachricht senden und ihnen mitteilen, dass sie das Modul über die direkte URL aufrufen sollen, z. B. http://www.ihre-seite.de/modules/news.

Um die Reihenfolge der Module zu ändern (was sich auf die Menüanzeige auswirkt), genügt es, die Module per Drag-and-Drop an die gewünschte Position zu verschieben.

Die Installation von Modulen erfordert etwas Planung – sowohl hinsichtlich der darzustellenden Inhalte als auch des Seitenlayouts. Es ist sinnvoll, sich andere XOOPS-Websites anzuschauen, um zu sehen, welche Module für welche Inhalte verwendet werden. Dies erkennt man leicht, indem man mit der Maus über die Menülinks in den Seitenleisten fährt. In der Statusleiste des Browsers wird dann die URL angezeigt, z. B. http://die-seite.de/modules/news/.... Daraus erkennt man, dass das Modul im Verzeichnis /modules/ liegt, und das nächste Segment der URL (hier „news“) das jeweilige Modul angibt.

Wenn Sie die Seite aufrufen, sehen Sie, wie das Modul eingesetzt wird und welche Art von Inhalt es darstellt. Das Vergleichen mit anderen XOOPS-Seiten gibt Ihnen zusätzlich Inspiration für Layout- und Designmöglichkeiten.
