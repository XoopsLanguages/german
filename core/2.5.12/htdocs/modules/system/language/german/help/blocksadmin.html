Hilfe: Blöcke

Beschreibung

Blöcke sind diskrete Inhaltsabschnitte, die in der Admin-Oberfläche erstellt und konfiguriert werden können. Benutzerdefinierte Blöcke können erstellt werden und bestehen typischerweise aus Text, Grafiken und Bildern. Die Inhalte dieser Blöcke können individuell formatiert werden oder die Formatierung der Website übernehmen. Blöcke können auch Teil der Anzeigeausgabe eines Moduls sein. Die Blockverwaltungsseite ist der Ort, an dem alle Blöcke konfiguriert und verwaltet werden.

Blöcke sind wie die Titelseite einer Zeitung: Sie sind Teaser, die Leser dazu anregen, vollständigere Geschichten auf den Innenseiten zu lesen. Blöcke bieten kleine Stücke dynamischen Inhalts mit Links zu den eigentlichen Inhalten, die im Modul gespeichert sind.

Ein Teil der Vorgänge, die im Hintergrund ablaufen, wenn ein Modul auf einer XOOPS-Website installiert wird, besteht darin, dass die Blöcke des Moduls im System registriert werden, sodass sie vom Webmaster nach Belieben verwendet werden können.

Was kann ein Webmaster mit einem Block tun? Nun, vor allem entscheiden, wo auf der Website der Block angezeigt wird (vergleichbar mit der Definition eines Schaufensters, das in diesem Fall draußen auf der Startseite oder sogar innen sein kann: Ein Block kann so definiert werden, dass er in anderen Modulen angezeigt wird), und wenn er angezeigt wird, wo auf einer Seite er zu sehen sein wird.

Jedes Modul, das auf einer XOOPS-Website installiert ist, registriert seine Blöcke im System, und der Webmaster kann entscheiden, wo sie angezeigt werden, in welchem Teil der Seite sie erscheinen und welche Relevanz, Gewichtung oder Position sie haben. Der Website-Administrator kann auch den Titel des Blocks bearbeiten und, wenn er ausreichend Kenntnisse hat, sogar die Vorlage bearbeiten, die der Präsentation des Blocks zugrunde liegt.

Tipps

Sie können den Titel, der angezeigt wird, wenn der Block dargestellt wird, ausblenden, indem Sie "// " (also Schrägstrich-Schrägstrich-Leerzeichen) als die ersten drei Zeichen des Titels hinzufügen. Auf diese Weise können Sie mehrere Blöcke nahtlos verbinden, um die gewünschte Ansicht zu erstellen.

Sie können Ihre eigenen Inhaltsblöcke mit der Schaltfläche "Block hinzufügen" erstellen. Sie können Ihre eigenen Inhalte in HTML, einfachem Text mit XOOPS-Codes oder sogar PHP erstellen. In Ihren benutzerdefinierten Blöcken wird der Code "{X_SITEURL}" durch die URL Ihrer Website mit einem abschließenden Schrägstrich ersetzt.
