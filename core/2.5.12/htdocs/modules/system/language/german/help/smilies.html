Hilfe: Smilies

Beschreibung

Smilies sind kleine grafische Symbole, mit denen Benutzer ihre Gefühle oder Reaktionen ausdrücken können. Sie können in jeden beliebigen Text eines Artikels, Kommentars oder Beitrags eingefügt werden – sowohl von der Benutzer- als auch von der Administratorseite aus. XOOPS wird standardmäßig mit einer Sammlung von 17 Smilies ausgeliefert. Einige davon werden direkt in den Eingabeformularen verschiedener Module angezeigt (z. B. Forum, News, Downloads, Links u. a.). Ein Klick auf ein Smilie fügt es direkt in den Beitrag ein.

Smilies lassen sich auch über ihre individuellen „Codes“ (Tastenkombinationen) einfügen. Wenn man z. B. „:“ „-“ „D“ eintippt, erscheint dieses Smilie:

Die vollständige Liste aller auf Ihrem System verfügbaren Smilies und ihrer Codes finden Sie im Smilie-Manager – genau hier. Dort können Sie Smilies hinzufügen, bearbeiten und löschen sowie festlegen, welche in den Eingabeformularen angezeigt werden (standardmäßig werden nur neun der 17 angezeigt).

Wenn Sie keine Smilies auf Ihrer Website verwenden möchten, können Sie einfach die Option „Im Formular anzeigen?“ deaktivieren – damit verschwinden sie aus den Eingabeformularen, bleiben aber über ihre Codes weiterhin verfügbar. Alternativ können Sie Smilies auch vollständig löschen.
