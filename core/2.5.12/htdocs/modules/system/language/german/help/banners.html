Hilfe: Banner

Beschreibung

Banner sind Werbeanzeigen, die im oberen Bereich der Seite angezeigt werden. Sie können aus Bildern oder Flash-Filmen bestehen, z. B. das sich heftig bewegende XOOPS-Banner, das Teil der Standardinstallation ist. XOOPS verfügt über ein integriertes System zur Verwaltung von Bannern und zur Erstellung von Werbekonten, das unter Systemmodul → Banner zu finden ist. Eine XOOPS-Website kann mehrere Banner anzeigen – der Anzeigebereich rotiert bei jedem Laden einer neuen Seite durch die aktiven Banner.

Jedes Mal, wenn ein Banner einem Besucher angezeigt wird, zählt dies als eine sogenannte „Impression“ – dies ist die gängige Handelswährung bei Bannerwerbung. Werbekunden zahlen in der Regel für eine bestimmte Anzahl an Impressionen – d. h. dafür, dass ihre Anzeige eine bestimmte Anzahl an Malen angezeigt wird.

Hinweis: Wenn Sie Bannerverkauf auf Basis von Impressionen betreiben, sollten Sie unbedingt Folgendes tun: Gehen Sie zu Systemmodul → Einstellungen → Allgemeine Einstellungen und tragen Sie Ihre eigene IP-Adresse in das Feld „Ihre IP-Adresse“ ein (sofern vorhanden). Dadurch wird verhindert, dass Ihre eigenen Seitenaufrufe in die Impression-Zählung einfließen. Wenn Sie dies nicht tun, könnte dies als unehrlich angesehen werden (wobei zu beachten ist, dass viele Administratoren keine feste IP-Adresse haben – z. B. bei Einwahlverbindungen).
