Installation von XOOPS

Erstmalige Installation

Vorwort:

XOOPS ist ein Open-Source, objektorientiertes Web-Publishing-System, das in PHP geschrieben wurde. Es ist ein ideales Werkzeug zur Entwicklung kleiner bis großer dynamischer Community-Websites, Intranet-Portale, Unternehmensportale, Weblogs und vieles mehr.

XOOPS wird unter den Bedingungen der GNU General Public License (GPL) Version 2 oder höher veröffentlicht und kann kostenlos genutzt und modifiziert werden. Es darf frei weiterverbreitet werden, solange die Verteilungsbedingungen der GPL eingehalten werden.

Anforderungen

  • WWW-Server (Apache, NGINX, IIS, etc.)
  • PHP 5.6.0 oder höher, 7.4+ empfohlen
  • MySQL 5.6 oder höher, MySQL 8.0+ empfohlen

Vor der Installation

  1. Richten Sie den WWW-Server, PHP und den Datenbankserver ordnungsgemäß ein.
  2. Bereiten Sie eine Datenbank für Ihre XOOPS-Website vor.
  3. Erstellen Sie ein Benutzerkonto und gewähren Sie dem Benutzer Zugriff auf die Datenbank.
  4. Machen Sie die Verzeichnisse und Dateien beschreibbar: %s
  5. Aus Sicherheitsgründen wird dringend empfohlen, die beiden folgenden Verzeichnisse aus dem Document Root zu verschieben und die Verzeichnisnamen zu ändern: %s
  6. Erstellen Sie (falls noch nicht vorhanden) und machen Sie die Verzeichnisse beschreibbar: %s
  7. Aktivieren Sie Cookies und JavaScript in Ihrem Browser.

Lokale Installation

Wenn Sie eine lokale Umgebung für Entwicklung oder Tests betreiben, stellen Sie sicher, dass die oben genannten Anforderungen erfüllt sind. Kopieren Sie anschließend den Inhalt des htdocs-Verzeichnisses (aus der XOOPS-Distributionsdatei) in den Stammverzeichnis-Pfad Ihrer Webumgebung. Sobald die Dateien kopiert sind, können Sie die Installation starten, indem Sie Ihren Browser auf "http://ihrewebsite.com" richten. Dies startet den Installationsprozess.

Installation auf einer gehosteten Plattform

Wenn Sie in einer gehosteten Umgebung arbeiten, entpacken Sie die XOOPS-Dateien lokal oder auf dem Server, falls Sie Shell-Zugriff (z. B. SSH) haben. Stellen Sie sicher, dass Sie alle XOOPS-Dateien aus dem htdocs-Verzeichnis in Ihr Stamm-Webverzeichnis verschieben oder kopieren (Ihr Anbieter gibt in der Regel diesen Speicherort sowie Anweisungen an). Sobald die Dateien kopiert sind, können Sie die Installation starten, indem Sie Ihren Browser auf "http://ihrewebsite.com" richten. Dies startet den Installationsprozess.

Fortsetzen der Installation

Folgen Sie einfach den Schritt-für-Schritt-Anweisungen im Installationsprogramm.
